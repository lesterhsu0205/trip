## 🤖 自動合併功能

本專案使用 GitHub Actions 自動合併各日行程檔案：

- 📝 編輯任何 `D*.md` 或 `todo.md` 檔案時
- 🔄 自動將所有子檔案合併到 `東京輕井澤行程總覽.md`
- 📱 GitHub 上查看時可展開完整內容
- 🔗 同時保留連結導航便於快速跳轉
